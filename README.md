# Объектно-ориентированное программирование: Вариант 12 (Лаб. №2)

### Описание
Проект посвящен углубленному изучению C++ и реализации семантики копирования/перемещения.

### Реализованные концепции
* **Правило трех/пяти**: Реализация конструктора копирования, оператора присваивания и деструктора для управления ресурсами.
* **Перегрузка операторов**: Создание интуитивно понятного интерфейса для работы с пользовательским классом.
* **Инкапсуляция**: Строгое разделение интерфейса (.h) и реализации (.cpp).

### Технологии
* C++ (Standard 14/17)
* Сборка: Makefile (или g++)
